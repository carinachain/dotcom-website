... 

This project is developed using the [next.js framework](https://nextjs.org/).

## Develop
1. Clone the project:
```shell
cd ~
git clone https://github.com/carinachain/dotcom-website.git
cd dotcom-website
```
2. Install the dependencies and run the dev server:
```shell
npm i
npm run dev
```
